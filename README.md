Projekat "Optimization of Pit Stop Decisions using the Fibonacci Heap Structure" radi na poboljšanju donošenja odluka o odlasku u box u Formuli 1 pomoću struktura podataka za upravljanje prioritetima.
Cilj projekta je pokazati kako se teorijska rješenja iz algoritama, poput Fibonacci heap-a, mogu primijeniti u realnim uslovima trke i procijeniti njihova stvarna efikasnost.

**Uputstvo za pokretanje**
Unutar foldera aplikacije otvoriti terminal i pokrenuti instalaciju:
pip install streamlit fastf1 pandas numpy plotly

Pokrenuti aplikaciju sa:
streamlit run f1simulation.py

ili sa:
python -m streamlit run f1simulation.py
